About Me

DevOps & Platform Engineer passionate about simplifying infrastructure

Hi, I'm Fidelis, a DevOps & Platform Engineer passionate about simplifying infrastructure and empowering development teams to move faster and more securely.

I specialize in designing and automating cloud infrastructure using AWS, Terraform, and CI/CD pipelines. My journey started with hands-on curiosity and has grown into architecting scalable environments, building reusable infrastructure modules, and integrating observability, compliance, and security from day one.

Over the past few years, I've worked on:

  • Creating modular Terraform stacks (VPC, ECR, PostgreSQL, SQS)
  • Developing serverless tools like an image resizer on AWS Lambda + API Gateway
  • Building secure CI/CD pipelines using GitHub Actions, SSM, CloudWatch, and SNS/PagerDuty
  • Supporting cloud-native applications with real-time monitoring, secret management, and secure network access (VPN, S3, IAM)

I'm currently building platforms that abstract complexity and offer self-service infrastructure for dev teams. Whether it's Terraform with Terragrunt or pipelines with GitHub Actions, I focus on making infrastructure repeatable, scalable, and developer-friendly.

Outside of work, I enjoy building open-source tools, writing technical walkthroughs, and mentoring those starting their cloud journey.
