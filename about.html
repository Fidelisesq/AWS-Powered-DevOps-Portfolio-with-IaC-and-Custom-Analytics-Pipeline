About Me

DevOps & Platform Engineer passionate about simplifying infrastructure

Hi, I'm Fidelis Ikoroje — a DevOps & Platform Engineer passionate about building resilient, secure, and developer-friendly infrastructure on the cloud. With a background that blends hands-on engineering with automation-first thinking, I specialize in architecting modular AWS environments using Terraform and Terragrunt, while enabling seamless, GitHub Actions-powered CI/CD across environments.

I've led the development of reusable infrastructure modules (VPC, ECR, RDS, SQS), enforced security best practices (OIDC, KMS, VPN, IAM), and integrated full-stack observability (CloudWatch, OpenTelemetry, Pingdom) to support critical payment services. My workflows prioritize scalability, compliance, and operational excellence — with DRY configs, automated testing, and measurable impact like reduced incident resolution time and faster developer onboarding.

I'm currently focused on building internal platforms that abstract complexity and provide self-service infrastructure for dev teams. Whether it's Terraform with Terragrunt or pipelines with GitHub Actions, I aim to make infrastructure repeatable, scalable, and developer-friendly.

Outside of work, I build hands-on projects to sharpen my skills and explore emerging technologies. Some recent projects include:

  • 2048 Game with AWS CloudFront & CI/CD – a fully serverless web game integrated with API Gateway, Lambda, DynamoDB, and Slack notifications.
  • Serverless Image Resizer – an on-demand image processing tool using S3, Lambda (Sharp), API Gateway, and CloudFront.
  • DeepSeek LLM Deployment – deployed an LLM (R1) on AWS EC2 behind an ALB with secure VPC, Route 53 DNS, and Terraform automation.
  • Cloud Resume Challenge – serverless portfolio project with a dynamic visitor counter, GitHub Actions CI/CD, and observability via CloudWatch and PagerDuty.

I also enjoy writing technical walkthroughs, contributing to open-source, and helping others break into cloud and DevOps careers.
